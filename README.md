# Crown System

Personal life tracking system with 3 pillars: Truth, Intelligence, and Creativity.

## Features
- Daily absolutes tracking (Prayer, Hygiene, Environment, Exercise, Reading, Languages)
- Project management for technical papers (INT_PA)
- Solvix initiative tracking (INT_AC)
- Knowledge library (TRUTH_AC)
- Art gallery for drawings and poetry
- Sports activity log
- Dark mode
- Weekly reflection and stats

## Tech Stack
- React
- TailwindCSS
- Supabase
- Vercel
